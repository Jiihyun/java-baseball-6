# 기능 목록

## BallNumbers

- 검증
    - 세자리 수 검증 []
    - 1 ~ 9 사이인지 검증 []
    - 숫자만 들어왔는지 검증 []
- 사용자가 입력한 숫자와 정답 숫자 비교
    - 같은 자리에 같은 수가 있는 지 비교 (스트라이크) []
    - 다른 자리에 같은 수가 있는 지 비교 (볼) []

## GameResult

- 입력한 수에 대한 결과를 볼, 스트라이크 개수 결과 알려주기 []
- 둘 다 개수 0 일시 낫싱 알려주기 []

## InputView

- 숫자 입력받기 []
- 게임 재시작 입력받기 []

## RamdomBallNumberGenerator

- 랜덤으로 1~9 사이 세자리 수 생성 (정답 역할) []

## GameController

- 3개 숫자 모두 맞히면 게임 종료 []
- 게임 종료 후 다시 시작(1) or 완전히 종료(2) []
